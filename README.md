# PrimeBets Demo

Local-first sports betting style demo built with React, Express, Tailwind CSS, and SQLite.

## Stack

- React + Vite frontend
- Express API backend
- SQLite database seeded with dummy data
- Tailwind CSS styling
- Fake local authentication with protected routes

## Run locally

1. Install dependencies:
   `npm install`
   `npm run install:all`
2. Start the app:
   `npm run dev`
3. Open:
   `http://localhost:5173`

## Demo accounts

- User: `demo@primebets.local` / `demo123`
- Admin: `admin@primebets.local` / `admin123`

## Project structure

- `client/`: frontend app
- `server/`: local API and SQLite database setup

## Notes

- This project is for localhost UI/UX testing only.
- No external services or payment providers are used.
- All app data is stored locally in SQLite.
